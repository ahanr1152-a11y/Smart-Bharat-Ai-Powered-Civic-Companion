# 🇮🇳 Smart Bharat Civic Companion

Smart Bharat Civic Companion is a frontend-based web application designed to simplify access to government services through an intuitive and user-friendly interface. The project aims to provide citizens with a single platform where they can explore government schemes, report civic issues, track complaints, and access essential public service information.

> 🚧 This is the initial frontend prototype. Backend integration and AI-powered features will be added in future updates.

---

## ✨ Features

- 🤖 AI Chat Assistant (Frontend Demo)
- 📋 Government Scheme Recommendation
- 📄 Document Requirement Assistant
- 🚨 Public Issue Reporting
- 📍 Complaint Tracking
- 🗺️ Nearby Government Office Finder
- 👤 User Profile
- 🌙 Dark Mode
- 📱 Fully Responsive Design
